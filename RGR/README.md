# Розрахунково-графічна робота
Дисципліна:«Бази даних та засоби управління»
Тема: Розробка додатку бази даних (PostgreSQL + Python)
Виконала: Студентка групи КВ-31, Галактіонова А.

### Опис проєкту
Консольний додаток для управління базою даних приватної клініки. Програма реалізована мовою Python з використанням архітектури MVC (Model-View-Controller). Додаток дозволяє керувати картками пацієнтів та лікарів, фіксувати візити, а також генерувати великі обсяги тестових даних та виконувати пошук за критеріями.

### Структура бази даних
База даних "Клініка" складається з трьох основних таблиць:

1. patient (Пацієнти)
   - Зберігає персональні дані: ПІБ, дата народження, email, стать.
2. doctor (Лікарі)
   - Зберігає дані медичного персоналу: ПІБ, email, спеціалізація.
3. visit (Візити)
   - Зв'язує пацієнта та лікаря, фіксує дату прийому та встановлений діагноз.

### Функціонал
Перегляд (Read): Виведення списку всіх лікарів клініки.
  Додавання (Create):Реєстрація нових пацієнтів.
  Видалення (Delete): Видалення лікаря з автоматичним контролем цілісності даних (заборона видалення, якщо є пов'язані візити).
  Генерація даних: Автоматичне створення 1000+ записів за допомогою оптимізованих SQL-запитів (використання `random()`, `generate_series` та масивів).
  Пошук: Фільтрація історії візитів за частиною назви діагнозу та діапазоном дат.

### Технології
Мова програмування: Python 3.10+
СУБД: PostgreSQL 17
Бібліотеки: `psycopg2-binary` (драйвер для роботи з БД)
Середовище розробки: Visual Studio Code
