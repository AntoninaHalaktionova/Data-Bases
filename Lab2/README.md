### Опис проекту
Предметна область: Платформа для зберігання історій хвороб пацієнтів.
База даних містить інформацію про пацієнтів, їх діагнози, візити.

### Виконання
1. Реалізовано об’єктно-реляційне відображення для моделей проекту.
2. Індекси: Додано індекси до таблиць для оптимізації пошукових запитів.
3. Тригери: Створено тригер для автоматизації дій при зміні даних.
4. Транзакції:
   * Створено тестову таблицю `medical_records` (Історії хвороб).
   * Наповнено таблицю даними.
   * Практично продемонстровано роботу транзакцій на рівні Read Committed.
   * Наведено код для аналізу рівнів Repeatable Read та Serializable.
